Linux: make install.
Mac: make install.
Build/Calc installation folder by default.

\section{Usage}
       
Two calculator modes are implemented:
 - Engineering calculator for calculations and graphing
 - Calculator for calculating loan payments
 - Deposit calculator

\section{Deletion}
